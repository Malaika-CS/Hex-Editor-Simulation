#  Hex Editor Simulation

##  Project Overview
A GUI-based Hex Decimal Editor first developed in C++ and then converted to COAL Assembly Language (.asm) and simulated on EMU8086 emulator. This project provides hex editing and visualization functionality.
## Objectives
- To implement a Hex Editor using C++ and Assembly Language
- To create GUI based interface in C++
- To convert and simulate the project on EMU8086
- To understand low level programming concepts
##  Key Concepts
- C++ Programming
- Assembly Language Programming
- Hexadecimal Number System
- GUI Development
- EMU8086 Simulation
- Low Level Programming
## Working Explanation
The Hex Editor was first developed as a GUI based application in C++. It was then converted to COAL Assembly Language (.asm file) and simulated using EMU8086 emulator. Users can view and edit data in hexadecimal format.
